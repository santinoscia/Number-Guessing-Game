Number Guessing Game


In this program, users must guess a number from 0-10 within a defined number of guesses. Increasing the difficulty increases the precision of the generated number. Users can replay the game after finishing.

Running the program will create a window that the game can be played in. No additional steps are required to run the program.


Variables that can be changed:

MIN_DIFFICULTY_LEVEL: The minimum difficulty that can be selected by the user. Default value = 1 (no decimal places).

MAX_DIFFICULTY_LEVEL: The maximum difficulty that can be selected by the user. Default value = 5 (4 decimal places).

MIN_ATTEMPTS: The minimum number of max attempts that can be selected by the user. Default value = 5.

MAX_ATTEMPTS: The maximum number of max attempts that can be selected by the user. Default value = 15.
